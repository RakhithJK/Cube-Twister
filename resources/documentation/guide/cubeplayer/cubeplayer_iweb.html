CubePlayer Applets

Usage in iWeb

You can use the following HTML Snippet to integrate a CubePlayer applet into an iWeb page:


<script type="text/javascript">
  var script="R-U2BL-FU-BDFUD-LD2F-RB-DF-U-B-UD-";
  if (navigator.userAgent.indexOf('iWeb')!=-1) {
    document.write('<div style="border:1px solid;width:270px;height:300px">'+script+'</div>');
  } else {
    document.write(
    '<applet mayscript="mayscript" '+
    '    codebase="http://www.randelshofer.ch/cube/lib" '+
    '    archive="RubikPlayerFlat.jar" code="RubikPlayerFlat" '+
    '    width="270" height="300"> '+
    '  <param name="script" value="'+script+'" /> '+
    '  <param name="codebase_lookup" value="false" /> '+
    '  <param name="classloader_cache" value="false" /> '+
    '  <param name="java_arguments" value="-Djnlp.packEnabled=true" /> '+
    '  <param name="permissions" value="sandbox" />'+
    '  <p>Install Java to see the cube.</p> '+
    '</applet>'
    );
  }
</script>
