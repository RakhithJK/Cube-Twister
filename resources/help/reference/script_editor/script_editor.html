The Script Editor

You can use the script editor to describe a transition between two states of the cube.

To open the script editor select a script in the tree.

The script editor consists of a three dimensional view of the cube and of a property sheet.

The cube view:

  • The 3D view
  • The «Reset» button
  • The «Twist» tool
  • The six «Paint Stickers» tools
  • The Popup Menu button

The property sheet:

  • The «Name» field
  • The «Reset» button
  • The «Record» button
  • The «Play» button
  • The Slider and the «Next» and «Previous» buttons
  • The «Check» button
  • The Popup Menu button
  • The «State» tab
  • The «Macros» tab
  • The «View» tab
  • The «Options» tab
  • The «Notes» tab

The solver uses a two-phase IDA* algorithm based on artificial intelligence (IDA* = Iterative Deeping search with non-overestimating heuristics). To reduce computation time, the solver uses tables containing precomputed twists and tables providing information about the distance from the current state of the cube to the solved state.
