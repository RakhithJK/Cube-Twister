The Script Editor - The Popup Menu

The popup menu of the script editor has the following items:

Cube

Use the Cube submenu to choose a Cube for the current script.

Notation

Use the Notation submenu to choose a notation for the current script.

Translate Into

Use this submenu to translate the current script into a different notation.

Reset Stickers

Resets the stickers of the cube. All stickers show the color defined by the current cube.

Clear Stickers

Removes all stickers from the cube except the ones for the six side parts.

Fill Stickers

Fills the remaining stickers with their default color.

Solve Cube

Uses the solver to create a script that solves the current state of the cube.

Generate Cube

Uses the solver to create a script that generates the current state of the cube.
