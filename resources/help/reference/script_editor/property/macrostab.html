The Script Editor - The «Macros» Tab

Use the «Macros» tab to specify local macros for the current script.

The «Popup» Menu

Using the popup menu you can add new macros, duplicate macros and delete macros.

The «Idenitfier» Field

Use this field to specify the identifier(s) of the macro.

If you enter multiple words into a field, then each word serves as an identifier for the macro.

Please note: Identifiers for macros must be unique per notation. If you specify the same identifier multiple times, then the behaviour of the script is undefined.

The «Script» Field

Use this field to specify the script of the macro.

The «Description» Field

Use this field to enter a description about the macro.
