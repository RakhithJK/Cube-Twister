The Script Editor - The «State» Tab

Use the «State» tab to see information about the script and about the current state of the cube.

The «Error» line

This line shows information about errors in the script.

This line is shown only, if the script contains error.

The «Twists» line

This line shows the twist count of the script. The twist count is shown in three different metrics:

  • The «Layer Turn Metric» is denoted wit the letter «l».
    In this metric, 90° and 180° layer turns are counted as 1.
  • The «Face Turn Metric» is denoted by the letter «f».
    In this metric, 90° and 180° face turns are counted as 1.
    Turns affecting other layers than the face layers are counted by the number of equivalent face turns.
  • The «Quarter Turn Metric» is denoted by the letter «q».
    In this metric, 90° face turns of the script are counted as 1, 180° face turns are counted as 2.
    Turns affecting other layers than the face layers are counted by the number of equivalent face turns.

This line is shown only, when the script has been checked successfully.

The «Order» line

This line shows the order of the current permutation of the cube. The order tells us, how many distinct states of the cube can be achieved by applying the current permutation repeatedly.

 

  • The «Visual Order Metric» is denoted with the letter «v».
    This metric ignores orientation changes of side parts.
  • The «Real Order Metric» is denoted by the letter «r».
    This metric takes orientation changes of side parts into account.

The «Permutation» line

This line shows the current permutation of the cube.
The permutation is shown using a sequence of disjoint cycles. For details, see the permutation notation.

The line appears only, when the cube is in a state, that can be described using a permutation.

Tip: If the current notation supports permutations, you can copy the contents of the permutation line into the script text field.

 

The «Stickers» line

This line shows the current position of the stickers on the cube.
The stickers are displayed using the sticker notation.

The line appears only, when the cube is not in a state, that can be described using a permutation.
