The Script Editor - The «View» Tab

Use the «View» tab to see an alternative view of the cube.

The «View» tab shows the rear of the cube. Together with the 3D view, you can see all faces of the cube at the same time.

All tools that are activated in the 3D view can also be applied to the rear view.

Tip: The rear view is useful when entering the state of a cube using the Paint Stickers tools,
