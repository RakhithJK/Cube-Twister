The Cube Editor - The «Parts» Tab

Use the «Parts» tab to change the properties of the cube parts.

Please note: The number of parts and their names depend on the current shape of the cube (see the «Shape» chooser).

The «Visible» Property

Use this property to show or hide a part in the 3D view.

(If you like, you can use the «Parts» tool as an alternative way for changing the visibility of a part.)

The «Name» Property

This property is used to name a part.

The name depends on the current shape of the cube. You can not change it.

The «Fill Color»Property

Use this property to associate the part with one of the colors in the «Colors» tab.

The «Outline Color»Property

Use this property to associate the part with one of the colors in the «Colors» tab.
