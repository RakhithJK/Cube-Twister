The Cube Editor - The «Colors» Tab

Use the «Colors» tab to specify the colors of the cube. You can add, remove and duplicate colors by using the popup menu. After you have added a new color, you can assign it to cube parts and stickers using the «Parts» tab and the «Stickers» tab.

Please note: If you remove a color, all parts and sticker that were assigned to this color are assigned to the next color on the «Colors» tab. The «Colors» tab must contain at least one color. You can delete all colors but one.

The «Color» Property

Use this property to specify the value of the color. (Clicking this property opens a color chooser dialog.)

The «Name» Property

Use the «Name» property to name the color. (A good name for a color would be a hint about its use in the «Parts» tab and the «Stickers» tab.)

The name is used for display in the «Parts» tab and in the «Stickers» tab.

The name should be unique per cube (but does not have to be).
