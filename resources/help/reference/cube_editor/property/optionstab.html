The Cube Editor - The «Options» Tab

The «Options» tab is used to specify the dimension and the orientation of the cube.

The «Dimension» properties

The «Dimension» properties specify the dimensions of the cube with respect to other cubes. This is useful if you want to create a cube design consisting of multiple cubes.

The «Scale» Slider

The «Scale» slider changes the scale factor of the cube in the range from 50 % to 200 %.

The «Explode» Slider

The «Explode» slider changes the explosion factor of the cube in the range from 0 % to 200 %.

The «Orientation» properties

The alpha and beta angles are used to specify the orientation of the cube when the «Reset» button is pressed.

The «Alpha» Slider

The «Alpha» slider changes the orientation of the cube along the x-axis.

The «Beta» Slider

The «Beta» slider changes the orientation of the cube along the y-axis.
