The Cube Editor - The «Stickers» Tab

Use the «Stickers» tab to change the properties of the stickers.

Please note: The number of stickers and their names depend on the current shape of the cube (see the «Shape» chooser).

The «Visible» Property

Use this property to show or hide a sticker in the 3D view.

(If you like, you can use the «Stickers» tool as an alternative way for changing the visibility of a part.)

The «Name» Property

This property is used to name a sticker.

The name depends on the current shape of the cube. You can not change it.

The «Fill Color» Property

Use this property to associate the sticker with one of the colors in the «Colors» tab.
