The Cube Editor

You can use the cube editor to specify the appearance of a cube.

To open the cube editor select a cube in the tree.

The cube editor consists of a three dimensional view of the cube and of a property sheet.

The cube view:

  • The 3D view
  • The «Reset» button
  • The «Twist» tool
  • The «Parts» tool
  • The «Stickers» tool

The property sheet:

  • The «Name» field
  • The «Shape» chooser
  • The «Reset» button
  • The «Use as default Cube» check box
  • The «Parts» tab
  • The «Stickers» tab
  • The «Colors» tab
  • The «Options» tab
  • The «Notes» tab
