The Notation Editor - The «Expressions» Tab

The «Expressions» tab provides a list of expression definitions that you can turn on and off. For each expression definition you can enter the admissible keywords into text fields. For some expressions you can also specify grammar rules.

Expressions:

  • Permutation
  • Grouping
  • Repetition
  • Inversion
  • Reflection
  • Conjugation
  • Commutation
  • Rotation
  • Delimiter
  • Comments
