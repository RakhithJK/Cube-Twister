The Notation Editor - Inversion

An inversion is used to specify an inversion of a statement. An inverted statement is done in reverse order and in the opposite turn direction.

Please note: Inversions are available only, if quarter turn face twists or permutations are enabled.

The «Inversion» checkbox:

Tick this checkbox to enable support for inversions by the notation.

When this checkbox is ticked, then additional fields are shown below the checkbox.

«Invertor» text fields:

Use this text field to specify the keyword that denote the invertor.

You must fill in this field, if inversions are enabled.

If you enter multiple words in this text field, then each word represents a keyword. The first word in the text field is used by CubeTwister for output of scripts in this notation.

«Position» radio buttons:

Use these radio buttons to specify the location of the invertor.

If the position is specified as «Leading», then the invertor is prepended to the statement. If the position is specified as «Trailing», then the invertor is appended to the statement.

Example 1:

If you fill in the text fields as shown in the table below, then the statement «A» is inverted by specifying «inverse A».

Invertor:  inverse 
Position:  leading

Example 2:

If you fill in the text fields as shown in the table below, then the statement «A» is inverted by specifying «A'».

Invertor:  ' 
Position:  trailing
