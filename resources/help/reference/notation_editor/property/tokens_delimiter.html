The Notation Editor - Delimiter

Delimiters are used to visually enhance the separation of statements.

The «Delimiter» checkbox:

Tick this checkbox to enable support for delimiters by the notation.

«Delimiter» text field:

Use these text fields to specify the delimiter to be used to separate each statement of a script.

«NOP» text field:

Use these text fields to specify the No Operation keyword. This keyword is used to separate sections of a script.
