The Notation Editor - Two-Layer Twists

-To Do-

A two-layer twist denotes a turn of two adjacent layers of the cube.

Please note: Two-layer twists are available only, if quarter turn face twists are enabled.

The «Two-Layer Twists» checkbox:

Tick this checkbox to enable support for two-layer twists by the notation.

When this checkbox is ticked, then entry fields for clockwise and anticlockwise quarter turn twists are shown below the checkbox.

If half turn face twists are in effect, then entry fields for clockwise and anticlockwise half turn twists are shown as well.

«Quarter Turn» text fields:

Use these text fields to specify the keywords for 90° two-layer wists.

You should specify at least one keyword for each face of the cube. For each face you can either fill in keywords for clockwise turns, anticlockwise turns or for both directions.

If you enter multiple words in a text field, then each word represents a keyword. The first word in a text field is used by CubeTwister for output of scripts in this notation.

«Half Turn» text fields:

Use these text fields to specify the keywords for 180° mid-layer wists.

You should specify at least one half turn keyword for each of the three axes of the cube.

Plase note: Half turn text fields are shown only, if half turn face twists are enabled.
