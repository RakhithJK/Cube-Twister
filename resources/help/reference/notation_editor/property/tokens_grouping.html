The Notation Editor - Grouping

A grouping is used to group statements.

Please note: Groupings are available only, if quarter turn face twists or permutations are enabled.

The «Grouping» checkbox:

Tick this checkbox to enable support for groupings by the notation.

When this checkbox is ticked, then two entry fields are shown below the checkbox.

«Grouping Begin» and «Grouping End» text fields:

Use these text fields to specify the keywords that denote the begin and the end of a grouping.

You must specify at least one keyword for each entry field.

If you enter multiple words in a text field, then each word represents a keyword. The first word in a text field is used by CubeTwister for output of scripts in this notation.

Please note: You may choose keywords that are ambiguous with permutation cycle begin and cycle end keywords only, if none of the twist keywords is ambiguous with one of the other permutation keywords.

Example 1:

If you fill in the text fields as shown in the table below, then the letter «(» denotes the beginning of a grouping, and the letter «)» the end of a grouping.

Grouping Begin:  ( Grouping End:   ) 

Example 2:

If you fill in the text fields as shown in the table below, then each of the letters «(», «[», «{» denotes the beginning of a grouping, and each of the letters «)», «]», «}» denotes the end of a grouping.

Grouping Begin:  ( [ { Grouping End:   ) ] )
