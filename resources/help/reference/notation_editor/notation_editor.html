The Notation Editor

You can use the notation editor to define your own notation to write down a script.

To open the notation editor select a notation in the tree.

The notation editor consists of a property sheet.

The property sheet:

  • The «Name» field
  • The «Use as default Notation» checkbox
  • The «Twists» tab
  • The «Expressions» tab
  • The «Macros» tab
  • The «Notes» tab
