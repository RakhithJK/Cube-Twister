Sticker Notation

The sticker notation is used to describe the position of the stickers on a Rubik's Cube.

 

CubeTwister currently supports only the following notation for stickers:

It consists of the name of a face, followed by a colon, and a list of sticker names. The name of a sticker is the name of the face, it is residing on, when the cube is solved.

Thus a solved cube is described using the following script:

F:FFFFFFFFF
R:RRRRRRRRR
D:DDDDDDDDD
B:BBBBBBBBB
L:LLLLLLLLL
U:UUUUUUUUU

The Pons Asinorum pretty pattern is described using the following script:

F:FBFBFBFBF
R:RLRLRLRLR
D:DUDUDUDUD
B:BFBFBFBFB
L:LRLRLRLRL
U:UDUDUDUDU
