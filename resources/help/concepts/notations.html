Notations

A notation defines the keywords and the grammar used to write a script.

Unfortunately there is no standard notation for Rubik's Cube. There are many conceptual and linguistic differences between notations.

You can write a script using one of the predefined notations provided with CubeTwister, or you can create your own notation(s) using the notation editor.

Notations are managed per document. To see the notations in the current document open the folder named «Notations» in the tree.
