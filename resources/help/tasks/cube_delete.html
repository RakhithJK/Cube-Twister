Deleting a Cube

Please note: You can not delete the default cube. If the cube you want to delete is the default cube, you have to make another cube the default before you can delete it.

To delete a cube:

  1. Select a cube in the «Cubes» folder of the tree.
  2. Choose «Delete» from the popup menu.
    (This will remove the cube from the tree, and assign the default cube to all scripts that used the cube you deleted.)
