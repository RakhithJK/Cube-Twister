Changing the Template

When you open a new CubeTwister window or when you choose «New» in the «File» menu, you do not start with an empty document. Instead CubeTwister loads a document called the «Template», which contains some predefined notations, cubes and script items.

To change the template on Mac OS X:

  1. Create a new CubeTwister document, and change it according to your needs.
  2. Choose «Save As» in the «File» Menu.
  3. Select the folder
    /Users/<your name>/Library/Preferences/CubeTwister
    and enter the filename template.xml into the file chooser.
  4. Press the «Save» button.

To change the template on Windows 95:

  1. Create a new document, and change it according to your needs.
  2. Choose «Save As» in the «File» Menu.
  3. Select the directory
    C:\Windows\Library\Preferences\CubeTwister
    and enter the filename template.xml into the file chooser.
  4. Press the «Save» button.
