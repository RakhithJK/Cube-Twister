Deleting a Notation

Please note: You can not delete the default notation. If the notation you want to delete is the default notation, you have to make another notation the default before you can proceed with the following steps.

To delete a notation:

  1. Select a notation in the «Notations» folder of the tree.
  2. Choose «Delete» from the popup menu.
    (This will remove the notation from the tree, and assign the default notation to all scripts that used the notation.)
