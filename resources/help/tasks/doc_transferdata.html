Transferring Data From One Document to Another

To transfer data from one document to another:

  1. Open the first document in CubeTwister using «Open» in the «File» menu.
  2. Choose «New Window» in the «File» menu.
    (This will open a new window.)
  3. Open the second document in the new window.
  4. Select the items you want to transfer in the tree of the first window.
  5. Choose «Copy» in the «Edit» menu of the first window.
  6. Select the tree of the second window.
  7. Choose «Paste» in the «Edit» menu of the second window.
