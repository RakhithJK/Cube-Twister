Creating a New Cube

To create a new cube:

  1. Choose «New Cube» from the «Edit» menu.
    (This will insert a new cube in the «Cubes» folder of the tree.)
  2. Open the «Cubes» folder in the tree.
  3. Select the newly created cube.
    (It has the name «unnamed Cube»).
  4. Click into the name to make it editable.
  5. Enter a new name for the cube.
  6. Press the «Enter» key to finish editing the name.
