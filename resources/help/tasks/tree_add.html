Adding an Item to the Tree

Please note: Each tree folder can only hold one type of items. For instance, if you try to create a new script in the «Notations» folder, it will be automatically added to the end of the «Scripts» folder.

To add an item to the tree:

  1. Select an item of the tree.
    (CubeTwister uses this item to determine the insertion point for the item you are adding.)
  2. Open the popup menu of the tree.
  3. Choose «New Cube», «New Notation», «New Script» or «New Note» in the popup menu.
    (This will add the new item after the one you have selected in step 1.)
