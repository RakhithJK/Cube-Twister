Creating a New Notation

To create a new notation:

  1. Choose «New Notation» from the «Edit» menu.
    (This will insert a new cube in the «Notations» folder of the tree.)
  2. Open the «Notations» folder in the tree.
  3. Select the newly created notation.
    (It has the name 'unnamed Notation'.)
  4. Click into the name to make it editable.
  5. Enter a new name for the notation.
  6. Press «Enter» to finish editing the name
