Changing the Default Cube

The default cube is automatically assigned to every newly created script.

To change the default cube:

  1. Open the «Cubes» folder in the tree.
  2. Select a cube that you want to become the default.
    (This will show the cube at the right of the tree as well a property sheet further right.)
  3. Tick the «Use as default» check box in the property sheet.
