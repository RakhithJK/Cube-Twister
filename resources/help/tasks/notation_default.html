Changing the Default Notation

The default notation is automatically assigned to every newly created script.

To change the default notation:

  1. Open the «Notations» folder in the tree.
  2. Select a notation that you want to become the default.
    (This will show a property sheet at the right of the tree.)
  3. Tick the «Use as default» check box in the property sheet.
