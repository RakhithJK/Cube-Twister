Arranging Tree Items

CubeTwister keeps the items in the tree always in the same sequence as they were added to the tree.

Please note: Each foldes can only hold one type of items. Thus you can change the sequence of the items only within one folder.

To arrange items in the tree:

  1. Select the items you want to move further down in a folder.
  2. Choose «Cut» in the popup menu of the tree.
    (This will remove the items from the tree and put them onto the clipboard.)
  3. Select the item after which you want to insert the items.
  4. Choose «Paste» in the popup menu of the tree.
    (Thils will read the items from the clipboard and insert them after the item you have selected in step 3.)
