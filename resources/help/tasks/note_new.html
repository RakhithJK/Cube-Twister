Creating a New Note

To create a new note:

  1. Choose «New Note» from the «Edit» menu.
    (This will insert a new note in the «Notes» folder of the tree.)
  2. Open the «Notes» folder in the tree.
  3. Select the newly created note.
    (It has the name «unnamed Note».)
  4. Click into the name to make it editable.
  5. Enter a new name for the note.
  6. Press the «Enter» key to finish editing the name
