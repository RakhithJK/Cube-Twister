Creating a New Script

To create a new script:

  1. Choose «New Script» from the «Edit» menu.
    (This will insert a new script in the «Script» folder of the tree.)
  2. Open the «Scripts» folder in the tree.
  3. Select the newly created script.
    (It has the name «unnamed Script».)
  4. Click into the name to make it editable.
  5. Enter a new name for the script.
  6. Press «Enter» to finish editing the name.
